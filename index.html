How It Works

이렇게 진행됩니다

체계적인 5단계 프로세스로 프로젝트를 완성합니다. 각 단계마다 명확한 산출물과 확인 절차가 있습니다.

1
무료 상담 카카오 / 전화
요구사항을 몰라도 됩니다. 대화하면서 정리해드립니다.
상담 후 24시간 내 초안 견적 제공
2
견적 · 범위 확정 24~48시간
숨겨진 추가 비용 없습니다. 계약 전 전체 금액을 확정합니다.
기능 목록 문서로 공유 — 나중에 "그런 말 했나요?" 없음
3
계약 + 개발 시작 착수금 50%
진행 상황을 주 1회 이상 공유합니다. 언제든 피드백 가능.
중간 결과물 프리뷰 링크 공유
4
납품 + 무상 수정 잔금 50%
납품 후 14일간 무상 수정. 소스코드 전체 전달.
배포까지 도와드립니다 (Vercel, AWS 등)

납품 후에도 계속 함께할 수 있습니다. 강요하지 않습니다.

FAQ

자주 묻는 질문

문의 전 먼저 확인해보세요.

개발 지식이 전혀 없어도 의뢰할 수 있나요? +
물론입니다. 오히려 비개발자 고객이 더 많습니다. 아이디어를 말씀해 주시면 어떤 방식으로 구현할지 쉽게 설명해드립니다. 기술 용어를 전혀 몰라도 됩니다.
납기를 못 지키면 어떻게 되나요? +
계약서에 납기를 명시하며, 지연 시 지연 일수만큼 추가 수정을 무상으로 제공합니다. 지금까지 납기를 어긴 사례가 없으며, 불가피한 상황 발생 시 즉시 고객에게 알려드립니다.
중간에 요구사항이 바뀌면 어떻게 되나요? +
소규모 변경은 유연하게 반영합니다. 범위가 크게 바뀌는 경우, 추가 비용과 일정을 미리 협의 후 진행합니다. 계약 범위 외 작업을 몰래 청구하는 일은 없습니다.
소스코드를 저한테 주나요? +
네, 기본으로 포함됩니다. 납품 시 GitHub 또는 압축 파일로 전체 소스코드를 전달합니다. 코드 소유권은 완전히 고객에게 있습니다.
계약금 구조는 어떻게 되나요? +
착수금 50%, 납품 후 잔금 50% 구조입니다. 장기 프로젝트의 경우 3단계 분할(착수 30% / 중간 40% / 납품 30%)도 협의 가능합니다.
유지보수는 어떻게 하나요? +
납품 후 14~30일(플랜별 상이)은 무상 수정입니다. 이후에는 월정액 유지보수 계약 또는 건별 요청 방식을 선택할 수 있습니다. 강요하지 않습니다.
